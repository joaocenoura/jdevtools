# jdevtools
A java development ecosystem built with vagrant and ansible

# quickstart
```
git clone https://github.com/joaocenoura/jdevtools.git .
vagrant up
```
after the installation process complete, go to:
- GitLab @ http://localhost:8080/
- Jenkins @ xxx
- Sonar @ xxx

And you're done!
